## Requirements
- Knowledge about BGA soldering
- Proper tools for BGA soldering
- BIOS Patch (available in Steam Deck Discord)

### LCD Steam Deck
- Modding to 32GB is most common
- 48GB might be possible?
- Tested Modules:
    - Samsung 8GB LPDDR5X K3LKCKC0BM-MGCP (x4)
